# Getting Started
* `git clone https://github.com/curationexperts/rangey.git`
* `cd rangey`
* `bundle install`
* `rake db:migrate`
* `bin/rails hyrax:default_admin_set:create`
* `bundle exec bin/rails hydra:server`
